# <a name="use-the-my-server-app-to-connect-to-windows-server-essentials"></a>Verwenden der My Server-App zur Verbindung mit Windows Server Essentials

>Gilt für: Windows Server2016 Essentials, Windows Server2012 R2 Essentials, Windows Server2012 Essentials

Die My Server-apps für Windows Server Essentials können Sie eine Verbindung mit Ressourcen herstellen und einfache administrative Aufgaben auf dem Windows Server Essentials-Server von Ihrem Laptop, PC oder Surface-Gerät ausführen. In My Server können Sie Benutzer, Geräte und Warnungen verwalten und mit freigegebenen Dateien auf dem Server arbeiten. Wenn Sie offline sind, können Sie diesen zum Arbeiten mit Dateien, die auf die in My Server zuletzt zugegriffen haben. Ihre offline erfolgten Änderungen werden automatisch mit dem Server das nächste Mal synchronisiert, das Sie eine Verbindung herstellen.  
  
 Wenn Ihr Server das Betriebssystem Windows Server Essentials ausgeführt wird, verwenden Sie die My Server-original-app. Wenn Ihr Server das Betriebssystem Windows Server Essentials ausgeführt wird, müssen Sie die aktualisierte My Server 2012 R2-app verwenden. Beide apps heruntergeladen werden können [Apps für Windows](https://windows.microsoft.com/windows-8/apps).  
  
> [!TIP]

>  Verwenden Sie für den Zugriff auf Ressourcen in Windows Server Essentials aus Ihrem Windows Phone die My Server Phone-app (für Windows Server Essentials) oder die My Server 2012 R2-Phone-app (für Windows Server Essentials). Informationen zur My Server Phone-app finden Sie unter [verwenden Sie die My Server-app für Windows Phone](Work-Remotely-in-Windows-Server-Essentials.md#BKMK_2). Informationen zur My Server 2012 R2-Phone-app finden Sie im Blogbeitrag [My Server 2012 R2 Windows und Windows Phone-apps](http://blogs.technet.com/b/sbs/archive/2013/11/19/my-server-2012-r2-windows-and-windows-phone-apps.aspx).

##  <a name="BKMK_WhatsNew"></a>Welche neuen in My Server 2012 R2 s  
 Zusätzlich zu den Features, die in My Server bereitstehen bietet My Server 2012 R2 die folgenden neuen Features für Kunden mit Windows Server Essentials:  
  
-   **Zugriff auf SharePoint Online-Teamwebsites und-Bibliotheken** -nutzen Sie Microsoft Office 365-Integration in Windows Server Essentials auf SharePoint Online-Teamwebsites zuzugreifen und Dokumente in Ihren SharePoint Online-Bibliotheken in My Server 2012 R2 zu öffnen.  
  
-   **Remotedesktopverbindungen zu den Server- und Clientcomputern** -Verwendung **Remoteverbindung** in My Server 2012 R2 zur Verbindung mit Ihrer Windows Server Essentials-Server und Clientcomputer mithilfe von Remotedesktop.  
  
##  <a name="BKMK_OS"></a>Betriebssystemanforderungen  
 Um My Server oder My Server 2012 R2 auf Ihrem Laptop, PC oder Surface-Gerät verwenden, die Computer benötigen Sie eine der folgenden Betriebssysteme: Windows 8.1, Windows RT 8.1, Windows 8 oder Windows RT.  
  
##  <a name="BKMK_Install"></a>Installieren von My Server  
 Sie können die entsprechende My Server-app aus dem [Apps für Windows](https://windows.microsoft.com/windows-8/apps) speichern. Wenn Ihr Computer, das Betriebssystem Windows 8 oder Windows RT verfügt und zur Verbindung mit des Servers im Intranet mit den Namen des Servers werden sollen, müssen Sie auch ein Zertifikat vom Server zu installieren.  
  
#### <a name="to-install-my-server-or-my-server-2012-r2-on-your-windows-based-laptop-pc-or-surface-device"></a>So installieren Sie My Server oder My Server 2012 R2 auf Ihrem Windows-basierten Laptop, PC oder Surface-Gerät  
  
1.  Installieren Sie die entsprechende My Server-app von [Apps für Windows](https://windows.microsoft.com/windows-8/apps) auf Ihrem Windows-basierten Laptop, PC oder Surface-Gerät.  
  
    |Server-Betriebssystem|Herunterladen|  
    |-----------------------------|--------------|  
    | Windows Server Essentials|[My Server](https://apps.microsoft.com/webpdp/app/19d94f81-db21-4234-8b49-806694dbbaea)|  
    | Windows Server Essentials|[My Server 2012 R2](https://apps.microsoft.com/webpdp/app/67e86695-bda3-4f32-96c4-2e20e56f1cf3)|  
  
2.  Wenn Sie den Namen des Servers zu verwenden, um mit dem Windows Server Essentials-Server über ein Intranet herstellen kann, und das Gerät, das Betriebssystem Windows 8 oder Windows RT verfügt, installieren Sie das Standardzertifikat "caroot.cer" auf Ihrem Computer durch Ausführen des folgenden Verfahrens. Manuelle Installation eines Zertifikats ist nicht in Windows 8.1 und Windows RT 8.1 erforderlich.  
  
###  <a name="BKMK_InstallCert"></a>Zum Installieren eines Zertifikats für My Server auf Ihrem Windows 8, Windows 8.1 oder Windows RT-Gerät  
  
1.  Öffnen Sie auf Ihrem Computer einen Webbrowser, und herunterladen Sie das Standardzertifikat "caroot.cer", vom Server. Hierzu geben Sie Folgendes ein, ersetzen Sie den Namen des Servers (z. B. **marketing.contoso.com**) für <*Servername*>:  
  
     **http://<servername\>/Connect/default.aspx?Get=caroot.cer**  
  
2.  Installieren Sie das Zertifikat:  
  
    1.  Auf der **starten** geöffnete Seite **Datei-Explorer**.  
  
    2.  Stellen Sie sicher, dass ausgeblendete Elemente und Dateinamenerweiterungen angezeigt werden: auf der **Ansicht** Registerkarte die **einblenden/ausblenden** Gruppe, und wählen die **ausgeblendete Elemente** und **Dateinamenerweiterungen** Kontrollkästchen.  
  
    3.  Navigieren Sie zu dem Ordner, der die Datei "caroot.cer" enthält, die Sie gerade heruntergeladen haben.  
  
    4.  Maustaste auf die Datei "caroot.cer", und klicken Sie dann auf **öffnen**.  
  
    5.  In der **Zertifikat** Dialogfeld, klicken Sie auf **Zertifikat installieren**.  
  
    6.  Wählen Sie **lokalen** als Installationsspeicherort aus, und klicken Sie dann auf **Weiter**.  
  
    7.  Auf der **Zertifikatspeicher** Assistentenseite wählen **alle Zertifikate in folgendem Speicher speichern**, und verwenden Sie **Durchsuchen** auswählen die **vertrauenswürdige Stammzertifizierungsstellen** speichern. Klicken Sie dann auf **Fertig stellen**.

Weitere Informationen finden Sie unter [zum Installieren eines Zertifikats für My Server auf Ihrem Windows 8, Windows 8.1 oder Windows RT-Gerät](#BKMK_InstallCert).)

##  <a name="BKMK_ConnectServer"></a>Wie Sie aus Ihrem lokalen Netzwerk mit dem Server herstellen  
 Um die My Server-app in Windows Server Essentials für Windows Phone, Windows 8 und Windows 8.1 erfolgreich zu verwenden, müssen Sie zunächst das Serverzertifikat auf Ihrem Gerät installieren. Dadurch wird das Gerät mit Ihrem Server unter Windows Server Essentials in Ihrem lokalen Netzwerk verbunden. Verwenden Sie die folgenden Verfahren an den Server in Ihrem lokalen Netzwerk herstellen und das Serverzertifikat auf Ihrem Windows Phone oder einem Computer mit Windows 8 oder Windows 8.1 installieren.  
  
#### <a name="to-connect-to-your-server-from-your-windows-phone"></a>Von Ihrer Windows Phone eine Verbindung mit dem server  
  
1.  Öffnen Sie auf Ihrem Windows Phone, auf denen Windows 8 oder Windows 8.1 ausgeführt wird, Internet Explorer.  
  
2.  Geben Sie in der Adressleiste **http://<yourservername\>/connect/default.aspx?get=caroot.cer**, und drücken Sie dann die EINGABETASTE.  
  
3.  Wenn Sie aufgefordert werden, die das Zertifikat "caroot.cer" zu installieren, klicken Sie auf **installieren**.  
  
4.  Warten Sie, bis zum Abschluss der Zertifikatinstallation.  
  
    > [!NOTE]
    >  Nach die Installation des Zertifikats abgeschlossen ist, können Sie die My Server-App für Windows Phone Anmelden mit Ihrem Servernamen und Ihren Netzwerkanmeldeinformationen.  
  
#### <a name="to-connect-to-your-server-in-your-local-network-from-a-computer-running-windows-8-or-windows-81"></a>Verbindung mit dem Server im lokalen Netzwerk auf einem Computer unter Windows 8 oder Windows 8.1  
  
1.  Öffnen Sie auf einem Computer, auf der Windows 8 oder Windows 8.1 ausgeführt wird Internet Explorer.  
  
2.  Geben Sie in der Adressleiste **http://<yourservername\>/connect/default.aspx?get=caroot.cer**, und drücken Sie dann die EINGABETASTE.  
  
3.  Wenn Sie aufgefordert werden, die das Zertifikat "caroot.cer" zu installieren, klicken Sie auf **öffnen**.  
  
4.  In der **Zertifikat** Dialogfeld, klicken Sie auf **Zertifikat installieren**.  
  
5.  Wählen Sie im Zertifikatimport-Assistenten, **lokalen** als Speicherort.  
  
6.  Wählen Sie **alle Zertifikate in folgendem Speicher speichern**, und wechseln Sie zu der **vertrauenswürdige Stammzertifizierungsstellen** Speicherort.  
  
7.  Führen Sie die Anweisungen, um den Assistenten abzuschließen.  
  
    > [!NOTE]
    >  Nach die Installation des Zertifikats abgeschlossen ist, können Sie die My Server-App für Windows 8 oder Windows 8.1 Anmelden mit Ihrem Servernamen und Ihren Netzwerkanmeldeinformationen.
