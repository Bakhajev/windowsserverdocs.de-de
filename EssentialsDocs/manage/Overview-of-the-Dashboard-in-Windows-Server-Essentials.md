# <a name="overview-of-the-dashboard-in-windows-server-essentials"></a>Übersicht über das Dashboard in Windows Server Essentials

>Gilt für: Windows Server2016 Essentials, Windows Server2012 R2 Essentials, Windows Server2012 Essentials
 
 Windows Server Essentials und Windows Server2012 R2 Standard mit aktivierter Windows Server Essentials Experience-Rolle enthalten ein Administrations-Dashboard, um die Aufgaben zu vereinfachen, die zum Verwalten von Windows Server Essentials-Netzwerk und dem Server ausführen. Mithilfe von Windows Server Essentials-Dashboard können Sie folgende Aktionen ausführen:  
  
-   Abschließen der Einrichtung des Servers  
  
-   Zugriff und allgemeine Verwaltungsaufgaben ausführen  
  
-   Serverwarnungen anzeigen und diesbezüglich Maßnahmen ergreifen  
  
-   Einrichten und Ändern von Einstellungen  
  
-   Suchen nach Hilfethemen im Web oder darauf zugreifen  
  
-   Community-Ressourcen im Web Access  
  
-   Verwalten von Benutzerkonten  
  
-   Geräte und Sicherungen  
  
-   Verwalten Sie Zugriff und Einstellungen für Serverordner und Festplatten  
  
-   Anzeigen und Verwalten von Add-In-Anwendungen  
  
-   In Microsoft online Services integrieren  
  
 Dieses Thema enthält:  
  
-   [Grundlegende Features des Dashboards](#BKMK_Design)

-   [Administrationsabschnitte des Dashboards](#BKMK_Features)

##  <a name="BKMK_Design"></a>Grundlegende Features des Dashboards  
 Windows Server Essentials-Dashboard können Sie schnell auf die wichtigsten Informationen und Verwaltungs-Features Ihres Servers zugreifen. Das Dashboard umfasst mehrere Abschnitte. Die folgende Tabelle beschreibt die Abschnitte.  
  
 
  
|Element|Dashboard-Feature|Beschreibung|  
|----------|-----------------------|-----------------|  
|1|Navigationsleiste|Klicken Sie auf einen Abschnittauf der Navigationsleiste auf die Informationen und Aufgaben, die dem Abschnittzugeordnet sind. Jedes Mal, die Sie das Dashboard öffnen, die **Home** Seite wird standardmäßig angezeigt.|  
|2|Unterabschnittsregisterkarten|Die unterabschnittsregisterkarten ermöglichen den Zugriff auf eine zweite Ebene von Windows Server Essentials-Verwaltungsaufgaben.|  
|3|Im Listenbereich|Die Listenansicht zeigt die Objekte, die Sie verwalten können, und grundlegende Informationen zu den einzelnen Objekten enthält.|  
|4|Im Detailbereich|Klicken Sie im Detailbereich zeigt zusätzliche Informationen über ein Objekt, das Sie in der Listenansicht auswählen.|  
|5|Aufgabenbereich|Der Aufgabenbereich enthält Links zu Tools und Informationen, mit denen Sie die Eigenschaften für ein bestimmtes Objekt (z.B. ein Benutzerkonto oder einen Computer) oder globale Einstellungen für die Objektkategorie zu verwalten. Der Aufgabenbereich ist in die folgenden zwei Abschnitte unterteilt:<br /><br /> **Aufgaben** œ enthält Links zu Informationen und Tools, mit denen Sie verwalten, die Eigenschaften für ein Objekt, das Sie in der Listenansicht (z.B. ein Benutzerkonto oder einen Computer) auswählen.<br /><br /> **globale Aufgaben** œ enthält Links zu Informationen und Tools, mit denen Sie globale Aufgaben für einen Feature-Bereich verwalten. Globale Aufgaben umfassen Aufgaben aus, um neue Objekte hinzuzufügen, Richtlinie festlegen und so weiter.|  
|6|Informationen und Einstellungen|Dieser Abschnittbietet direkten Zugriff auf die servereinstellungen, und einen Hilfelink zu Informationen über die Dashboardseite, die Sie anzeigen.|  
|7|Warnungsstatus|Der Status der Benachrichtigungen bietet eine rasche Übersicht über die Integrität des Servers. Klicken Sie auf das warnungsbild, um kritische und wichtige Warnungen anzuzeigen.<br /><br /> **Hinweis:** In Windows Server Essentials und Windows Server2012 R2 Standard mit installierter Windows Server Essentials Experience-Rolle, der Warnungsstatus finden Sie die **Informationen und Einstellungen** Registerkarte.|  
|8|Statusleiste|Die Statusleiste zeigt die Anzahl der Objekte, die angezeigt werden, in der Liste angezeigt. Add-In-Anwendungen können auch andere Statusanzeigen liefern.|  
  
##  <a name="BKMK_Home"></a>Features der Dashboard-Startseite angezeigt.  
 Wenn Sie das Dashboard Öffnen der **Home** Seite angezeigt wird, werden standardmäßig mit der **SETUP** Kategorie angezeigt. Die **Home** Seite von Windows Server Essentials-Dashboard bietet schnellen Zugriff auf die Aufgaben und Informationen, die unterstützt Sie beim Anpassen des Servers und konfigurieren Features wichtiger. Die Startseite besteht aus vier Funktionsbereichen, die Informationen und Konfigurationsaufgaben für die Optionen verfügbar zu machen, die Sie auswählen. Die folgende Tabelle beschreibt die Features.  
  
|Element|Funktion|Beschreibung|  
|----------|-------------|-----------------|  
|1|Navigationsleiste|Klicken Sie auf einen Abschnittauf der Navigationsleiste auf die Informationen und Aufgaben, die dem Abschnittzugeordnet sind. Jedes Mal, die Sie das Dashboard öffnen, die **Home** Seite werden standardmäßig angezeigt.|  
|2|Bereich Kategorie|Dieser Bereich zeigt Feature-Bereiche, die schnellen Zugriff auf Informationen bereitstellen und Configuration Tools, mit denen Sie anpassen und den Server. Klicken Sie auf eine Kategorie, um anzuzeigen, Aufgaben und Ressourcen, die dieser Kategorie zugeordnet sind.|  
|3|Aufgabenbereich|In diesem Bereich werden Links zu Aufgaben und Informationen zu einer ausgewählten Kategorie angezeigt. Klicken Sie auf eine Aufgabe oder Ressource, um weitere Informationen anzuzeigen.|  
|4|Klicken Sie im Aktivitätsbereich|Dieser Bereich enthält eine kurze Beschreibung eines Features oder einer Aufgabe und enthält Links, die Konfigurationsassistenten und Informationsseiten geöffnet. Klicken Sie auf einen Link, um weiteren Maßnahmen ergreifen.|

##  <a name="BKMK_AccessDb"></a>Zugriff auf Windows Server Essentials-Dashboard  
 Sie können das Dashboard zugreifen, mithilfe einer der folgenden Methoden. Die von Ihnen gewählte Methode hängt davon ab, ob Sie den Zugriff auf sind das Dashboard vom Server, auf einem Computer, der mit dem Windows Server Essentials-Netzwerk verbunden ist oder von einem Remotecomputer.  
  
 Um ein sicheres Netzwerk aufrecht zu erhalten, können nur Benutzer mit Administratorberechtigungen für die Windows Server Essentials-Dashboard zugreifen. Darüber hinaus können Sie das integrierte Administratorkonto über das Launchpad auf dem Windows Server Essentials-Dashboard anmelden.  
  
###  <a name="BKMK_Server"></a>Zugriff auf das Dashboard vom Server  
 Bei der Installation von Windows Server Essentials erstellt der Installationsvorgang eine Verknüpfung mit dem Dashboard auf die **starten** Bildschirm, und auch auf dem Desktop. Wenn die Verknüpfung von diesen Speicherorten nicht vorhanden ist, können Sie mithilfe der **Suche** Bereich zu suchen, und führen Sie das Dashboard aus.  
  
##### <a name="to-access-the-dashboard-from-the-server"></a>Zum Zugriff auf das Dashboard vom Server  
  
-   Melden Sie sich an den Server als Administrator, und führen Sie dann eine der folgenden:  
  
    -   Auf der **starten** auf **Dashboard**.  
  
    -   Doppelklicken Sie auf dem Desktop auf **Dashboard**.  
  
    -   In der **Suche** Bereich, Typ **Dashboard**, und klicken Sie dann auf **Dashboard** in den Suchergebnissen angezeigt.  
  
###  <a name="BKMK_Network"></a>Zugriff auf das Dashboard von einem Computer, der mit dem Netzwerk verbunden ist  
 In Windows Server Essentials nach dem Beitritt eines Computers mit dem Netzwerk können Administratoren das Launchpad Sie den Computer im Server-Dashboard zugreifen.  
  
> [!WARNING]
>  In Windows Server Essentials zum Herstellen einer Verbindung zum Dashboard vom Clientcomputer aus, mit der rechten Maustaste des Symbols, und öffnen Sie das Dashboard über das Kontextmenü.  
  
##### <a name="to-access-the-dashboard-by-using-the-launchpad"></a>Auf das Dashboard zugreifen, mit dem Launchpad  
  
1.  Öffnen Sie auf einem Computer, die mit dem Netzwerk verbunden ist, die **Launchpad**.  
  
2.  Klicken Sie im Launchpad auf **Dashboard**.  
  
3.  Auf dem Dashboard **anmelden** Seite Geben Sie Ihre Anmeldeinformationen als Netzwerkadministrator, und drücken Sie dann die EINGABETASTE.  
  
     Eine Remoteverbindung mit Windows Server Essentials-Dashboard wird geöffnet.  
  
###  <a name="BKMK_Remote"></a>Zugriff auf das Dashboard von einem Remotecomputer  
 Wenn Sie von einem Remotecomputer arbeiten, können Sie Windows Server Essentials-Dashboard zugreifen, mithilfe der Remotewebzugriff-Website.  
  
##### <a name="to-access-the-dashboard-by-using-remote-web-access"></a>Zum Zugriff auf das Dashboard mit Remote Web Access  
  
1.  Öffnen Sie auf dem Remotecomputer einen Internetbrowser, z.B. Internet Explorer.  
  
2.  Geben Sie in der Adressleiste des Internetbrowsers, Folgendes:**https://<DomainName\>/remote**, wobei *Domänenname* ist der Name der externen Domäne Ihrer Organisation (z.B. contoso.com).  
  
3.  Geben Sie Ihren Benutzernamen und das Kennwort zum Anmelden bei der Remotewebzugriff-Website.  
  
4.  In der **Computer** Teil der Remotewebzugriff**Home** Seite; Suchen Sie den Server, und klicken Sie auf **verbinden**.  
  
     Eine Remoteverbindung mit dem Dashboard wird geöffnet.  
  
    > [!NOTE]
    >  Wenn der Server nicht, in angezeigt wird der **Computer** Abschnittder Startseite, klicken Sie auf **Verbindung mit weiteren Computern**, suchen Sie den Server in der Liste, und klicken Sie dann auf dem Server herstellen.  
    >   
    >  So erteilen Sie eine Benutzer die Berechtigung zum Zugriff auf das Dashboard über den Remotewebzugriff auf: Öffnen Sie die Eigenschaftenseite für das Benutzerkonto ein, und wählen Sie dann die **Serverdashboard** Option auf der **"Zugriff überall"** Registerkarte.  
  
##  <a name="BKMK_UseSafeMode"></a>Verwenden des abgesicherten Modus  
 Das Feature Abgesicherter Modus von Windows Server Essentials überwacht die Add-Ins, die auf dem Server installiert sind. Wenn das Dashboard instabil wird oder nicht mehr reagiert wird, im Abgesicherter Modus erkennt und isoliert der Add-Ins, die möglicherweise das Problem verursachen, und zeigt sie auf die **Einstellungen des abgesicherten Modus** Seite das nächste Mal öffnen Sie das Dashboard. Von der **Einstellungen des abgesicherten Modus** Seite, die Sie deaktivieren und aktivieren Sie nacheinander zu bestimmen, welches Add-In das Problem verursacht-Add-Ins können. Sie können dann auswählen, um das Add-In bei zukünftigen Starts deaktiviert zu lassen, oder Sie können das Add-In deinstallieren.  
  
 Sie können eine Liste aller Add-Ins anzeigen, die auf dem Server zu einem beliebigen Zeitpunkt installiert sind. Sie können auch das Dashboard im abgesicherten Modus öffnen, die automatisch alle Nicht-Microsoft-Add-Ins deaktiviert. Windows Server Essentials können Sie auf das Dashboard im abgesicherten Modus von einem anderen Computer im Netzwerk zugreifen.  
  
#### <a name="to-view-a-list-of-installed-add-ins"></a>Zum Anzeigen einer Liste der installierten Add-Ins  
  
-   Klicken Sie auf dem Dashboard auf **Hilfe**, und klicken Sie dann auf **Einstellungen für abgesicherten Modus**.  
  
#### <a name="to-open-the-dashboard-in-safe-mode"></a>Öffnen Sie das Dashboard im abgesicherten Modus  
  
-   In der **Suche** Bereich, Typ **sicher**, und klicken Sie dann auf **Dashboard (Abgesicherter Modus)** in den Suchergebnissen angezeigt.  
  
#### <a name="to-open-the-dashboard-in-safe-mode-from-another-computer-on-the-network"></a>Öffnen Sie das Dashboard im abgesicherten Modus von einem anderen Computer im Netzwerk  
  
1.  Auf einem Computer, der mit dem Netzwerk verbunden ist, öffnen Sie das Windows Server Essentials-Launchpad, und klicken Sie dann auf **Dashboard**.  
  
2.  Geben Sie auf der Dashboard-Anmeldeseite den Benutzernamen und das Kennwort für ein Konto mit der Berechtigung zum Anmelden an den Server, wählen die **zulassen, dass ich Add-Ins zum Laden auswählen**, und klicken Sie dann auf den Pfeil, um anmelden.
