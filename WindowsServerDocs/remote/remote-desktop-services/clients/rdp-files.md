# <a name="supported-remote-desktop-rdp-file-settings"></a>Unterstützt die RDP-dateieinstellungen für Remotedesktop

Die folgende Tabelle enthält die Liste der unterstützten RDP-Datei-Einstellungen, die Sie mit den Windows- und HTML-Clients verwenden können. Ein "X" in der Plattform-Spalte gibt an, dass die Einstellung unterstützt wird. Diese Liste ist jedoch nicht über eine vollständige Liste der unterstützten Einstellungen für die Windows und HTML5-Clients. Wir werden weiterhin die folgende Tabelle enthält weitere unterstützte RDP-Einstellungen für die Windows und HTML5-Clients als auch das Mac OS, iOS und Android-Clients aktualisieren.
