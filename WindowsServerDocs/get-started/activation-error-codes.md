# Problembehandlung bei der Aktivierung-Fehlercodes

**Home-Benutzer** Dieser Artikel ist für die Verwendung von Support-Agents und IT-Experten vorgesehen. Wenn Sie weitere Informationen zu Windows-Aktivierung Fehlermeldungen suchen, wechseln Sie zu den folgenden Microsoft-Websites:

[Hilfe bei Aktivierungsfehlern in Windows](https://support.microsoft.com/help/10738/windows-10-get-help-with-activation-errors) 

Wenn Sie versuchen, ein Mehrfachaktivierungsschlüssel (MAK) oder (Key Management Service, KMS) verwenden, um die Volumenaktivierung für einen oder mehrere Windows-basierten Computer ausführen, erhalten Sie Fehlermeldungen, die bestimmten Fehlercodes enthalten. In diesem Artikel wird erläutert, wie Sie diese Fehler zu beheben.
