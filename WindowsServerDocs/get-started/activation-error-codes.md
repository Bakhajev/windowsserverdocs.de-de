# <a name="troubleshooting-activation-error-codes"></a>Problembehandlung bei Aktivierungsfehlercodes

**Heimanwender** dieser Artikel ist für die Verwendung von Supportmitarbeitern und IT-Spezialisten gedacht. Wenn Sie weitere Informationen zu Fehlermeldungen für Windows-Aktivierung suchen, wechseln Sie zu den folgenden Microsoft-Websites:

[Hilfe zu Fehlern beim Aktivieren von Windows](https://support.microsoft.com/help/10738/windows-10-get-help-with-activation-errors) 

Wenn Sie versuchen, einen Mehrfachaktivierungsschlüssel (MAK) oder den Schlüsselverwaltungsdienst (KMS) verwenden, um die Volumenaktivierung auf einen oder mehrere Windows-basierten Computern ausführen, erhalten Sie Fehlermeldungen, die bestimmten Fehlercodes enthalten. In diesem Artikel wird erläutert, wie Sie diese Fehler zu beheben.
