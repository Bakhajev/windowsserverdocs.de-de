# <a name="troubleshooting-activation-error-codes"></a>Behandeln von Problemen mit Aktivierungsfehlercodes

**Hinweis für Privatbenutzer:** Dieser Artikel ist für Supportmitarbeiter und IT-Spezialisten gedacht. Weitere Informationen zu Fehlermeldungen im Zusammenhang mit der Aktivierung von Windows findest du auf der folgenden Microsoft-Website:

[Hilfe für Fehler bei der Aktivierung von Windows](https://support.microsoft.com/help/10738/windows-10-get-help-with-activation-errors) 

Wenn du mithilfe eines Mehrfachaktivierungsschlüssels (Multiple Activation Key, MAK) oder des Schlüsselverwaltungsdiensts (Key Management Service, KMS) eine Volumenaktivierung auf einem oder mehreren Windows-basierten Computern ausführen möchtest, erhältst du Fehlermeldungen mit spezifischen Fehlercodes. In diesem Artikel erfährst du, wie du diese Fehler behandelst.
