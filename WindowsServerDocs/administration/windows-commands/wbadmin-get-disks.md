# <a name="wbadmin-get-disks"></a>Wbadmin Get-Datenträger



Listet die internen und externen Datenträger, die derzeit für den lokalen Computer online sind.

Um die Datenträger aufgelistet, die mit diesem Unterbefehl online sind, Sie müssen Mitglied der **Sicherungs-Operatoren** Gruppe oder der **Administratoren** Gruppe, oder Sie wurde die entsprechenden Berechtigungen delegiert. Darüber hinaus müssen Sie ausführen **Wbadmin** eine Eingabeaufforderung mit erhöhten Rechten. (Öffnen Sie eine Eingabeaufforderung mit erhöhten Rechten mit der rechten Maustaste **Eingabeaufforderung**, und klicken Sie dann auf **als Administrator ausführen**.)

## <a name="syntax"></a>Syntax

```
wbadmin get disks
```

## <a name="parameters"></a>Parameter

Dieses Unterbefehl hat keine Parameter.
