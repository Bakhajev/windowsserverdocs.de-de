# <a name="create"></a>erstellen



Erstellt eine Partition auf einem Datenträger, einem Volume auf einem oder mehreren Datenträgern oder eine virtuelle Festplatte (VHD).

## <a name="syntax"></a>Syntax

```
create partition
create volume
create vdisk
```
