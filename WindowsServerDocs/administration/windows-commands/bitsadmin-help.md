# <a name="bitsadmin-help"></a>bitsadmin Hilfe



Zeigt die Verwendung der Befehlszeile an.

## <a name="syntax"></a>Syntax

```
bitsadmin /help | /?
```

## <a name="BKMK_examples"></a>Beispiele

Das folgende Beispiel ruft die Befehlszeilen Hilfe ab.
```
C:\>bitsadmin /help
```
